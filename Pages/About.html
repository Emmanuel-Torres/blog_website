El Rincón del Blog

Project Description:

El Rincón del Blog is my Final Roject for CS 1810. This website consists of a webpage capable of storing and displaying blogs. The user is able create and submit a blog. After the user submits a post, this newly added post will be showcases along the existing ones on the home page of the website.

  • Bootstrap

Discussions:

Discussion about Accessibility

Accessibility is something crucial that should be incorporated in more websites. Adding various types of accessibility options to a website could be very beneficial to the website itself. Accessibility options will facilitate the user experience for users that have disabilities. In addition, it would raise the placement of the website in a web search result. Because it raises the website in the list of results, it will indirectly increase the traffic of users through the website, which could result in more revenue/profit.

Discussion about Search Engine Optimization

Search Engine Optimization is the process of designing and optimizing a website in such a way that it will appear higher on the list of results of a search engine. This is something convenient, because the higher our page is in a search result list, the more likely that users will reach our website. If our page is well optimized, it will get an increase of traffic, given that it will place high on the list. This type of optimization is achieved in various ways: having meaningful and semantic HTML, having integrated accessibility options, providing a good user experience/interface, providing share-worthy information that can increase the number of external references, etc. By doing some of these, we are guaranteed that our website will start placing higher. This is the main reason why Search Engine Optimizations is very important.

Discussion about JavaScript Transpiling

JavaScript is a language that has been around for many years, it is present in a great number of websites, and it is a language that is continually growing and changing. Due to its rapid changes, many websites that are not updated as often will soon start running outdated versions of JavaScript. Because of this rapid growth and change of the language, it makes sense why transpiling code is something necessary. Transpiling JavaScript is the process of taking current JavaScript code, and translating it into an older version of JavaScript. This process gives us the capability to run and use websites that are using legacy JS code.

Discussion about Static Web Generation

Static web generation happens when we take an HTML file and render it on the browser. Because HTML is a text markup language, it is not dynamic. A static webpage has fixed content that will not change regardless of what the user does. This is the main reason why static web generation is not that great. In order to change a statically generated site, you will need to manually manipulate and edit the original file. Modern websites don’t work like that. Today we have interactive websites that will change their content based on things like user interaction. This makes static web pages look obsolete. Nevertheless, today we still use static files to build the main structure of a page, and then we will make that static file dynamic through the use of languages like JavaScript. Incorporating JavaScript to a static file will allow the content of that static file to change without having us to manually make those changes.
